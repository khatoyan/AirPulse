# AirPulse - Мониторинг пыльцы и аллергенов

Приложение для мониторинга распространения пыльцы и аллергенов, помогающее людям с аллергией планировать свою активность.

## Возможности

- Интерактивная карта распространения аллергенов
- Календарь цветения растений
- Информация о различных видах аллергии
- Тесты на определение аллергии
- Предупреждения о периодах активного цветения

## Установка и запуск

### Предварительные требования

- Node.js (v16+)
- PostgreSQL
- npm или yarn

### Установка

1. Клонируйте репозиторий:
```
git clone https://github.com/yourusername/airpulse.git
cd airpulse
```

2. Установите зависимости для бэкенда и фронтенда:
```
# Бэкенд
cd backend
npm install

# Фронтенд
cd ../frontend
npm install
```

3. Настройте базу данных:
   - Создайте базу данных PostgreSQL
   - Скопируйте `.env.example` в `.env` и настройте подключение к базе данных

4. Выполните миграции и сгенерируйте клиент Prisma:
```
cd backend
npm run prisma:migrate
npm run prisma:generate
```

5. Заполните базу данных примерами растений для календаря цветения:
```
npm run seed:plants
```

### Запуск приложения

1. Запустите бэкенд:
```
cd backend
npm run dev
```

2. Запустите фронтенд:
```
cd frontend
npm run dev
```

3. Откройте приложение в браузере: `http://localhost:3000`

## Структура проекта

- `/backend` - серверная часть на Node.js + Express + Prisma
- `/frontend` - клиентская часть на React + Material UI
- `/docs` - документация

## Технологии

### Бэкенд
- Node.js и Express
- Prisma ORM
- PostgreSQL
- JWT для аутентификации

### Фронтенд
- React
- Material UI
- Leaflet для карт
- Axios
